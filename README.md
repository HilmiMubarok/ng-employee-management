# Employee Management Backoffice

Aplikasi web *responsive* untuk mengelola data karyawan, dibangun menggunakan Angular. Proyek ini mengimplementasikan fitur autentikasi sederhana, manajemen daftar dengan *pagination*, *sorting*, *filtering*, serta reactive form dengan validasi kompleks.

## Environment yang Digunakan
* Node.js (v18.x atau lebih baru disarankan))

## Cara Menjalankan Aplikasi Lokal

1. **Clone Repository**
   ```bash
   git clone https://github.com/HilmiMubarok/ng-employee-management.git
   cd ng-employee-management

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Application**
   ```bash
   npm start
   ```

Aplikasi akan berjalan di `http://localhost:4200`
